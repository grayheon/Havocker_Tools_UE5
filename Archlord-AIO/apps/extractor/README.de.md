# 📦 Extractor

Extrahiert und entschlüsselt alle relevanten Quelldateien gemäß Konfigurationspfad.

## Funktionen
- Kopiert reguläre Dateien (z. B. INI, TXT, XML, BMP, PNG, etc.)
- Erkennt verschlüsselte Formate automatisch
- Entschlüsselt `.ini`, `.txt`, `.xml` (sofern nicht übersprungen)
- Erkennt `.dat` / `.ref` Dateien, gibt diese an weitere Verarbeitung
