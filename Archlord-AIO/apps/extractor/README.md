# 📦 Extractor

Extracts and decrypts all relevant source files based on the configured paths.

## Features
- Copies regular files (e.g. INI, TXT, XML, BMP, PNG, etc.)
- Automatically detects encrypted formats
- Decrypts `.ini`, `.txt`, `.xml` (if not skipped)
- Passes `.dat` / `.ref` files to further processing
