# 🧩 OBJ Checker

Validates `.ini`-based object lists and checks referenced TID and DFF entries.

## Features
- Parses `objecttemplate.ini` (EUC-KR)
- Extracts relevant fields (TID, DFF, COLLISION, etc.)
- Checks whether referenced DFFs exist
- Creates:
  - CSV with object data
  - TXT reports (TID/DFF found or missing)
