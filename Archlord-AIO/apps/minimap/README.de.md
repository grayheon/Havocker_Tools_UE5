# 🗺️ Minimap Generator

Erzeugt aus vorhandenen Tile-Grafiken eine zusammengesetzte Weltkarte.

## Funktionen
- Sucht alle gültigen `mapXXXXa/b/c/d.*`-Dateien
- Erzeugt 2×2 Tiles (64 Minikacheln = 1 Map)
- Fügt alle Tiles zu einem großen Weltkartenbild zusammen
- Exportiert das Resultat als PNG oder BMP
